# Arizona (AZ) - Public Official Transparency Profiles

## 🚨 CROWDSOURCING CALL TO ACTION
Welcome to the Arizona node of the Lazarus Politician Portfolio Tracker. 

**MANDATE:** We are mandating absolute public transparency on the wallets, portfolios, and financial disclosures of every single elected official in the state of Arizona. The public has an unalienable right to audit their public officials. 

### Strict Zero-Simulation Policy
This directory contains **REAL DATA ONLY**. No mock data, no simulations, and no inferred values are permitted. 

### Authentication Checkpoints
Every politician profiled in this directory must pass an authentication checkpoint. This means their identity, office, and financial disclosures must be verified by **at least three (3) independent public resources**. 

**How You Can Help:**
1. **Verify Profiles:** Cross-reference the names generated by our spiders with the Official Arizona State Legislature website, Ballotpedia, and OpenSecrets.
2. **Contribute Disclosures:** If you find PDF or XML financial disclosures for AZ state officials, submit a Pull Request to append them to the specific politician's profile.
3. **Future Elections:** As new officials are elected or re-elected, this repository will automatically generate new tracking files. We need crowdsourced auditors to maintain the integrity of these incoming data streams.

Transparency is not requested; it is required. Join the audit.
