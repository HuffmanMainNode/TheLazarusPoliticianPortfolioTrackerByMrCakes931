# Database Schema

## Tables
- **politicians**: Stores basic information about politicians.
- **issuers**: Stores information about traded companies/assets.
- **trades**: Stores individual trade records.
- **committees**: Stores committee information.
- **politician_committees**: Association table mapping politicians to committees.
- **red_flags**: Stores detected anomalies and potential corruption indicators.
- **politician_scores**: Stores aggregated corruption scores.
