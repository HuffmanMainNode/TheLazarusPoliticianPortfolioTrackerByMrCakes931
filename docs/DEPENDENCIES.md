# Project Dependencies

## Core Python Libraries
- `requests`: For making HTTP requests to external APIs and data sources.
- `pandas`: For data manipulation, analysis, and validation queries.
- `SQLAlchemy`: The ORM used for database interactions and schema management.
- `yfinance`: For fetching live financial data and ticker metadata from Yahoo Finance.
- `beautifulsoup4`: For parsing HTML during web scraping operations (e.g., Senate disclosures).
- `pytest`: For the testing framework.
- `rich`: For enhanced console output and formatting.

## External APIs & Data Sources
- **Capitol Trades API**: For congressional trading data.
- **QuiverQuant API**: For house and senate trading data.
- **eHouse & eSenate Portals**: For official financial disclosure XML/PDF feeds.
- **Yahoo Finance**: For real-time asset metadata and ticker resolution.

## System Requirements
- **Python**: 3.10+
- **SQLite**: 3.0+
